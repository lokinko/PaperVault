## :jack_o_lantern: 项目简介

PaperVault 是一个用于收集和检索人工智能领域学术论文的全自动化工具，覆盖自然语言处理、计算机视觉、机器学习、数据挖掘、数据库、语音、系统、安全、网络、理论计算机科学等多个方向的顶级学术会议与期刊。
