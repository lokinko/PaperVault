## :jack_o_lantern: 项目简介

PaperVault 是一个用于收集和检索人工智能领域学术论文的全自动化工具，覆盖自然语言处理、计算机视觉、机器学习、数据挖掘、数据库、语音、系统、安全、网络、理论计算机科学等多个方向的顶级学术会议与期刊。

## 🚧 项目状态

> **本项目仍在积极施工中。**

<!-- auto-summary-start -->

- 数据库已收录 **340,000+** 篇论文，覆盖 NLP、CV、ML、DM、DB、Speech 等 117+ 个顶级会议与期刊。

### 下阶段目标

- 重新部署并上线 Web 搜索服务，支持更高效的论文检索与浏览。
